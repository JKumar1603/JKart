JKart E-Commerce Platform - Test Cases

Project Name:
JKart E-Commerce Platform - Multi-Vendor Marketplace

Purpose:
This document contains functional test cases for the JKart E-Commerce Platform. The test cases cover authentication, product management, admin approval, customer shopping, cart, order placement, payment, invoice, notification logging, and role-based access.
